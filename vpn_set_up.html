::Setting up OpenVPN Client on Linux to connect with IITB OpenVPN Server:: Back


Currently OpenVPN configuration is not working on Ubuntu 15.x series and Fedora 23 due to some bug.

Please follow bellow given instructions to install and configure OpenVPN client as per your Operating System.
    Installation for Ubuntu/Debian Users:
    -------------------------------------------
  1. To install openvpn client please use "Ubuntu Software Center" or "Synaptic Package Manager" which is available on your system.
    OR
    You can manually install it by typing the command 'sudo apt-get install openvpn ' in a Terminal.

  2. After successful client installation download the client configuration file as per your Operating System (32bit or 64bit)

    ubuntu_32bit_openvpn_client.tgz
    ubuntu_64bit_openvpn_client.tgz

  3. Now exract this file to "/etc/openvpn" directory, by overwriting all existing default config files. If you were using a terminal then use following command to extracts the file. 
     sudo cd /etc/openvpn
 sudo tar xzf "path_to_client_config_file"/xxx_yyy_openvpn_client.tgz 
		
	Where	(XXX --> fedora/ubuntu)
		(YYY --> 32bit/64bit)
    The following are the contents of the client configuration file. 
    	iitbvpn.conf
	client.up
	client.down
  4. To connect to the VPN server, open terminal. Then you needs to start the openvpn daemon using following command.

    Command for Ubuntu 14.x and Ubuntu 15.x Users:

     
 
	[user@ubuntu ~]$ sudo service openvpn start
	* Starting virtual private network daemon(s)...
	*	Autostarting VPN 'client'                                           
	Enter Auth Username: "USE YOUR LDAP ID"
	Enter Auth Password: "USE YOUR LDAP PASSWORD"
						
	[user@ubuntu ~]$
    Command for Ubuntu 16.x Users:

     
 
	[user@ubuntu ~]$ sudo systemctl start openvpn@iitbvpn.service
	* Starting virtual private network daemon(s)...
	*	Autostarting VPN 'client'                                           
	Enter Auth Username: "USE YOUR LDAP ID"
	Enter Auth Password: "USE YOUR LDAP PASSWORD"
						
	[user@ubuntu ~]$

    Now your machine is a part of the IITB network and you can access any intranet service of IITB network.

  5. To disconnect OpenVPN Service ,you needs to type following command in a terminal

    Command for Ubuntu 14.x and Ubuntu 15.x Users:

     
 
	[user@ubuntu ~]$ sudo service openvpn stop
	* Stoping virtual private network daemon(s)...
	*	Stoping VPN 'client'                [ OK ]
						
	[user@ubuntu ~]$
    Command for Ubuntu 16.x Users:

     
 
	[user@ubuntu ~]$ sudo systemctl stop openvpn@iitbvpn.service 
	* Stoping virtual private network daemon(s)...
	*	Stoping VPN 'client'                [ OK ]
						
	[user@ubuntu ~]$

    Installation for Fedora/RHEL/Centos Users:
    --------------------------------------
  1. To install 'openvpn' package either use 'Add/Remove Programs' gui tool.
    OR
    you can manually install it by typing this command 'yum install openvpn' or 'dnf install openvpn' in a Terminal as a root user.

  2. After successful client installation download the client configuration file as per your Operating System (32bit or 64bit)

    fedora_32bit_openvpn_client.tgz
    fedora_64bit_openvpn_client.tgz

  3. Now exract this file to '/etc/openvpn' directory, by overwriting all existing default config files. If you were using a terminal then login as root user then use following command to extracts the file. 
    	[root@fedora ~]# cd /etc/openvpn
	[root@fedora openvpn]# tar xzf "path_to_client_config_file"/xxx_yyy_openvpn_client.tgz
    The following are the contents of the file 
    	iitbvpn.conf
	client.up
	client.down
  4. To connect to the VPN server, again open terminal as a root user. Then you needs to start the openvpn daemon using following command.

    Command on older versions of Fedora, RHEL 5.X, RHEL 6.X and Centos 5.X, Centos 6.X Users:

     
 
	[root@fedora ~]# service openvpn start
	Starting openvpn:                                           
	Enter Auth Username: "USE YOUR LDAP ID"
	Enter Auth Password: "USE YOUR LDAP PASSWORD"
							[  OK  ]
	[root@fedora ~]#
    Command on latest versions of Fedora18 to Fedora 21, RHEL 7.X, and Centos 7.X Users:

     
	 
	[root@fedora ~]# systemctl start openvpn@iitbvpn.service
	Starting openvpn:                                           
	Enter Auth Username: "USE YOUR LDAP ID"
	Enter Auth Password: "USE YOUR LDAP PASSWORD"
	[root@fedora ~]#

    Now your machine is a part of the IITB network and you can access any intranet service of IITB network.

  5. To disconnect OpenVPN Service ,you needs to type following command in a terminal

    Command on older versions of Fedora, RHEL 5.X, RHEL 6.X and Centos 5.X, Centos 6.X Users:

    	[root@fedora ~]# service openvpn stop
	Shutting down openvpn:                           [  OK  ]
	[root@fedora ~]#
    Command on latest versions of Fedora18 to Fedora 21, RHEL 7.X, and Centos 7.X Users:

     
	[root@fedora ~]# systemctl stop openvpn@iitbvpn.service
	[root@fedora ~]#
